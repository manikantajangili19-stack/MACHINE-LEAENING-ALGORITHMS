NumPy Module 1:
 What is NumPy?
Why NumPy exists

Python lists are flexible, but for numerical data they are slower and less convenient.

NumPy gives:

faster operations
easy mathematical calculations
multi-dimensional arrays
support for matrix-like computations
base for Pandas, scikit-learn, ML libraries

====================================================================================================
Learn

converting 1D → 2D
converting 2D → 1D


arr = np.array([1,2,3,4,5,6])
arr.reshape(2,3)

output:
    [[1,2,3],
 [4,5,6]]

Must understand

If total elements = 12, reshape must still use 12 elements:

(3,4) valid
(2,6) valid
(5,2) invalid

                                                              NumPy Module 6: Broadcasting
Learn

Broadcasting means NumPy can do operations between arrays of different shapes when compatible.

Examples
add a number to every element
add one row vector to each row
multiply every value by 10

Why important

You avoid loops and write fast code.
