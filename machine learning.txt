                                                                    Machine Learning (ML)
----------------------------------------------------------------------------------------------------------------------------------------------------------------------

What is Machine Learning?

Machine Learning (ML) is a branch of Artificial Intelligence (AI) that enables computers to learn from data and make predictions or decisions without being explicitly programmed for every rule.

Simple Definition:_

Machine Learning is teaching a computer to learn from examples instead of writing all the rules yourself.

Real-Life Example:-
-----------------------

Imagine you want to teach a child to recognize a cat.

Traditional Programming:-
-------------------------

You tell the computer every rule:

Cat has 2 eyes
Cat has 4 legs
Cat has whiskers
Cat has a tail
Cat says "Meow"

If one rule changes, the program may fail.

Machine Learning:-
-----------------

Instead of writing rules, you show the computer 10,000 cat pictures and 10,000 dog pictures.

The computer learns the differences by itself.

Now when it sees a new picture, it predicts:

"This is a Cat."

This is Machine Learning.

1. Supervised Learning
------------------------------------------

Definition:-

The computer learns using labeled data.

Labeled means the correct answer is already known.

| House Size (X) | Price (Y) |
| -------------: | --------: |
|           1000 |  40 Lakhs |
|           1200 |  50 Lakhs |
|           1500 |  70 Lakhs |

The computer learns the relationship between size and price.

Now you ask:

 | What is the price of a 1300 sq ft house?

The model predicts it.

Used For

House Price Prediction
Salary Prediction
Student Marks Prediction
Spam Detection
Disease Prediction
-----------------------------------------------------------------------------------
                           Types of Supervised Learning
----------------------------------------------------------------------------------
Regression:-
--------------

Predicts a number.

Examples:

House Price
Salary
Temperature
Sales

output:-
 ₹72,50,000
-----------------------
Classification:-
--------------------

Predicts a category.

Examples:

Spam / Not Spam
Cat / Dog
Cancer / No Cancer
Pass / Fail

Output:
Spam
-------------------------------------------------------------------------------------------------------------------
                                             2. Unsupervised Learning
--------------------------------------------------------------------------------------------------------------------

Definition

The computer learns using unlabeled data.                                 

There are no correct answers.

It tries to find hidden patterns.

Example:

A shopping website has customer data but doesn't know who belongs to which type of customer.

The algorithm groups similar customers.

Result:

Group A → Students

Group B → Families

Group C → Business Customers

Nobody told the computer these groups—it discovered them.

Used For
Customer Segmentation
Fraud Detection
Recommendation Systems
Pattern Discovery

types of unsupervised learning:-

clustering
dimensionality
reduction
association rule learning
----------------------------------------------------------------------------------------
                                          3. Semi-Supervised Learning
----------------------------------------------------------------------------------------------
Definition

Uses a small amount of labeled data and a large amount of unlabeled data.

Example:

You have:

100 labeled medical images
10,000 unlabeled medical images

The model learns from both.

Used For
Medical Imaging
Speech Recognition
Face Recognition
---------------------------------------------------------------------------------------------------------
                                              4. Reinforcement Learning
---------------------------------------------------------------------------------------------------------------
Definition

The computer learns by trial and error.

It receives:

Reward for good actions
Penalty for bad actions

Example:

Teaching a dog:

Sit → Reward 🍖
Jump incorrectly → No reward

The dog learns what actions lead to rewards.

Similarly, an AI agent learns the best actions over time.

Used For

Self-Driving Cars
Robots
Chess
Video Games
Resource Optimization

Types of Reinforcement Learning

│
└── 4. Model-Free Reinforcement Learning
       │
       ├── Value-Based Learning

1. Supervised Learning
--------------------------------------
Used when you have input (X) and correct output (Y).


Supervised Learning learns from labeled data (inputs with correct outputs).
Features (X) are the inputs to the model.
Target (Y) is the output the model should predict.
The typical workflow is: Collect Data → Clean Data → Split Data → Train → Test → Predict.
There are two main tasks:
Regression → predicts numbers.
Classification → predicts categories.
---------------------------------------------------------

                Regression
----------------------------------------------------------

Regression is a type of Supervised Learning.
It learns from labeled data (inputs with correct numeric outputs).
The output is always a continuous numerical value.
The model finds patterns in historical data and uses them to predict new values.

| Algorithm                         | Easy Explanation                                               | Best Use Case                                             |
| -------------------------------   | -------------------------------------------------------------- | --------------------------------------------------------- |
| 1)Linear Regression               | Fits a straight line through the data                          | House price prediction                                    |
| 2)Multiple Linear Regression      | Uses multiple input feature                                    | Salary prediction using experience, education, and skills |
| 3)Polynomial Regression           | Fits a curved line instead of a straight line                  | Population growth                                         |
| Ridge Regression                  | Linear regression with L2 regularization to reduce overfitting | High-dimensional datasets                                 |
| Lasso Regression                  | Performs feature selection using L1 regularization             | Selecting important features                              |
| Elastic Net regularization        | Combines Ridge and Lasso                                       | Complex datasets                                          |
| Decision Tree (Regression Tree)   | Predicts values using tree-like decisions                      | Real estate pricing                                       |
| Random Forest                     | Combines many decision trees                                   | More accurate predictions                                 |
| Support Vector Regression (SVR)   | Uses support vectors for regression                            | Small to medium datasets                                  |
| K-Nearest Neighbors Regression    | Predicts using nearby data points                              | Local prediction problems                                 |
| Gradient Boosting                 | Builds models sequentially to improve accuracy                 | Sales forecasting                                         |
| XGBoost                           | Fast and highly accurate boosting algorithm                    | Competitions and industry                                 |
| LightGBM                          | Faster boosting for large datasets                             | Big data                                                  |
| CatBoost                          | Handles categorical data efficiently                           | Customer analytics                                        |

                                              Classification
---------------------------------------------------------------------------------------------------------------------------------------------------------------
Classification is a type of Supervised Learning.
It predicts categories (labels) rather than numbers.
It learns from labeled training data.
There are three common types:
Binary Classification (2 classes)
Multi-Class Classification (more than 2 classes)
Multi-Label Classification (multiple labels for one input)

| Algorithm                          | Easy Explanation                                           | Best Use Case                             |
| ---------------------------------- | ---------------------------------------------------------- | ----------------------------------------- |
| Logistic Regression                | Predicts the probability of each class                     | Spam detection, pass/fail prediction      |
| K-Nearest Neighbors (KNN)          | Looks at the nearest similar examples to classify new data | Handwriting recognition                   |
| Decision Tree                      | Makes decisions using a tree of questions                  | Loan approval                             |
| Random Forest                      | Combines many decision trees for better accuracy           | Disease prediction                        |
| Support Vector Machine (SVM)       | Finds the best boundary between different classes          | Face recognition                          |
| Naive Bayes classifier             | Uses probability to classify data                          | Email spam filtering                      |
| Gradient Boosting                  | Sequentially improves weak models                          | Customer churn prediction                 |
| XGBoost                            | Fast, powerful boosting algorithm                          | Fraud detection, competitions             |
| LightGBM                           | Optimized for large datasets                               | Large-scale classification                |
| CatBoost                           | Works well with categorical features                       | Customer analytics                        |
| Artificial Neural Network (ANN)    | Learns complex patterns                                    | General classification tasks              |
| Convolutional Neural Network (CNN) | Specialized for images                                     | Image classification                      |
| Recurrent Neural Network (RNN)     | Handles sequential data                                    | Text and speech classification            |
| Transformer                        | Advanced model for language understanding                  | Chatbots, translation, sentiment analysis |

                                            "How do I know which algorithm to use for a given dataset?"

   Step 1: Look at the Target (Y)

  Ask yourself:
|"What am I trying to predict?"
          There are four possibilities.

                     Start
                      │
        What do you want to predict?
                      │
      ┌───────────────┼───────
      │               │               │
 Number?         Category?      No Target?
      │               │               │
Regression      Classification    Clustering

Case 1: Output is a Number → Regression:-
-----------------------------------------------------------
| Input       | Output         |
| ----------- | -------------- |
| House Size  | ₹25 Lakhs      |
| Experience  | ₹60,000 Salary |
| Temperature | 32°C           |
| Area        | Crop Yield     |

Since the output is a number, use Regression algorithms.

Examples:

Linear Regression
Decision Tree Regression
Random Forest Regression

Case 2: Output is a Category → Classification:-
----------------------------------------------------------
| Input    | Output               |
| -------- | -------------------- |
| Email    | Spam                 |
| Student  | Pass                 |
| Customer | Buy / Not Buy        |
| Patient  | Disease / No Disease |

The output is a label, so use Classification algorithms.

Examples:

Logistic Regression
Decision Tree
Random Forest

Case 3: No Target Column → Clustering
---------------------------------------------------------
Suppose your dataset is:
| Customer | Age | Salary |
| -------- | --- | ------ |
| A        | 20  | 25000  |
| B        | 45  | 90000  |
| C        | 21  | 27000  |

There is no target column.

You only want to group similar customers.

Use:

K-means clustering

------------------------------------------------------------------
                 Do you have labels (Y)?
                        │
             ┌──────────┴──────────┐
             │                     │
            YES                    NO
             │                     │
       What is Y?             Group Similar Data
             │                     │
      ┌──────┴──────┐              │
      │             │              │
   Number       Category      K-Means Clustering
      │             │
Regression     Classification

Rule You Should Memorize

| If the target (Y) is...                  | Use...         |
| ---------------------------------------- | -------------- |
| A number (e.g., ₹50,000, 35°C)           | Regression     |
| A category (e.g., Yes/No, Spam/Not Spam) | Classification |
| No target column                         | Clustering     |
